本项目用作练习js，有许多js小案例，可供以后工作需要

总体分为五个目录
    - html      存放h5页面
    - css       存放css样式
    - js        存放js文件
    - img       存放图片
    - plugins   存放引用到外部的插件

****************************每次添加案例都需要在此写出案例说明****************************

1.top
    点击按钮，让页面回到顶部

2.mouseMove
    鼠标移入，字体改变颜色；鼠标移出，字体恢复颜色

3.scroll
    点击目录，页面滑动至目录所在处

4.browser
    使用cookie记录用户点击记录

5.fiveSecondReturnIndex
    5秒后返回某个页面

6.rightClick
    右击，在右击处弹出菜单（菜单样式比较酷炫）

7.carouselFigure
    轮播图效果（按钮透明化）

8.dynamicChange
    div随滚动条而滚动，两个div并行放置

9.menu
    鼠标移入一级菜单，显示其二级菜单，隐藏其他二级菜单；移出则隐藏二级菜单
